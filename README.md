# shopping-cart-with-JS

Un carrito de compras hecho con HTML CSS BOOSTRAP Y JS

Para utilizarlo solo se debe dar en el boton "buy" del articulo deseado, registrarse e iniciar sesion para poder completar la transacción

Por defecto apareceran los articulos ordenados por final, indicando nombre, cantidad y precio.

El carrito tambien otorgara la posibilidad de aumentar o reducir la cantidad del producto por fila tanto como se desee (afectando por su puesto el precio por producto, y el precio total).

De nesecitar vaciar todo el carrito se pulsa en el boton "empty"
